# 任务
请基于输入内容，输出一份结构化论文解读，默认面向初学者。

## 输入
- 论文主题：[论文主题]
- 论文内容：[论文内容]
- 读者水平：[读者水平]
- 输出重点：[输出重点]

## 要求
1. 首次出现的专业术语必须解释清楚。
2. 先讲“论文解决了什么问题，为什么重要”。
3. 再讲“方法怎么做、数据怎么来、指标怎么看”。
4. 实验结果不只复述，要给出启发和局限。
5. 给出可复现建议（最小实验路径）。

## 输出结构
1. 论文基本信息
2. 研究背景与核心贡献
3. 方法拆解（含关键术语解释）
4. 数据集与评估指标
5. 实验结果与解读
6. 局限性与可改进点
7. 给初学者的学习路径（3-5条）
