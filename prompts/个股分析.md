# Analysis Framework (Must Follow Strictly)
请严格按照以下五个模块输出，字数不少于2500字：

## 1. 核心结论与交易指引 (The Alpha)
*   **目标价模型**：必须给出具体的**目标市值（亿元）**及**对应股价（元）**。
    *   *要求*：根据行业属性选择合适的估值方法（科技股用PEG/SOTP，周期股用PB/ROE，消费股用DCF）。拒绝笼统，必须展示简化的计算逻辑（如：板块A利润 x 倍数 + 板块B利润 x 倍数）。
*   **核心逻辑（One-Liner）**：用一句话概括驱动股价上涨/下跌的最核心矛盾（如：困境反转、新产品放量、甚至戴维斯双击）。
*   **交易指令**：
    *   评级：强力买入 / 谨慎增持 / 观望 / 卖出
    *   点位：给出精确的**左侧低吸位**（基于支撑线/成本线）和**右侧追涨位**（基于突破口）。
    *   *止损红线*：逻辑证伪的硬止损位。

## 2. 深度基本面：颗粒度与非共识 (Deep Dive & Variant Perception)
*   **财务透视**：列出最新季度的核心数据（营收、净利、扣非增速），并点评质量（现金流如何？应收账款是否恶化？）。
*   **业务拆解（关键）**：
    *   **寻找"第二增长曲线"**：对于有新业务的公司，必须量化其进展（如：产能规划、客户定点、ASP提升、市场份额）。
    *   **存量业务**：是现金牛还是拖油瓶？
*   **非共识推演（Variant Perception）**：
    *   **市场的一致性预期错在哪里？**（例如：市场过于线性外推了高增长？还是忽视了某项隐形资产的重估？）
    *   **寻找"未被定价的X因素"**：挖掘一个市场忽略的利好或利空（如：竞争对手暴雷、管理层股权激励解锁动力）。

## 3. 筹码博弈与四维验证 (Market Structure & Cross-Validation)
请分析**谁在买，谁在卖**，并绘制共振矩阵：
*   **筹码结构**：
    *   **对手盘分析**：近期的抛压来自哪里？（定增解禁？游资获利了结？原始股东减持？）
    *   **潜在买家**：是否有机构/社保/ETF开始建仓？
    *   **关键成本线**：测算主力或定增机构的持仓成本，判断其护盘意愿。
*   **技术面**：当前处于什么阶段（吸筹/拉升/派发）？关键均线形态如何？
*   **资金面**：近期主力资金（机构/北向/游资）的流向特征。
*   **消息面**：未来3个月的时间轴（Time-Line），列出具体的催化剂日期（如：财报日、行业大会、新工厂投产）。

## 4. 极限压力测试 (Stress Test)
*   **清算价值测试**：若核心逻辑（如新业务订单）**完全归零**，仅靠存量业务，股价底线是多少？（计算"破净底线"或"安全边际"）。
*   **杀逻辑风险**：是否存在颠覆性的风险？（如：技术路线被替代、大客户剔除供应链）。

## 5. 情景推演 (Scenario Analysis)
*   **乐观情景（Upside）**：若核心假设超预期，股价上限在哪里？
*   **中性情景（Base）**：按部就班，股价的合理区间。
*   **悲观情景（Downside）**：结合上述压力测试，给出一个极端的防守价格。

# Constraints & Tone
1.  **拒绝定性，强制定量**：不要说"大幅增长"，要说"同比增长40%"；不要说"产能即将释放"，要说"预计12月底投产，新增产能XX万吨"。
2.  **数据时效性**：优先搜索并引用**最近两周**的公告、研报、新闻和交易数据。
3.  **信息源策略**：重点关注交易所公告（实锤）、雪球/通过深度文章（逻辑）、龙虎榜（资金）。
4.  **格式**：使用Markdown，重点数据加粗，逻辑清晰。
