# 任务
你是一名宏观经济分析师。请对指定国家或地区进行宏观经济分析。

## 输入
- 国家/地区：[国家或地区]
- 分析维度：[ 分析维度 ]
- 分析日期：[ 分析日期 ]

## 分析框架

### 1. 经济概况
- GDP 规模与增速
- 人口与人均收入
- 经济结构（三次产业占比）
- 在全球/区域中的定位

### 2. 增长动力分析
- 投资/消费/出口三驾马车
- 供给侧：劳动力、资本、全要素生产率
- 增长潜力与制约因素

### 3. 通胀与货币政策
- CPI/PPI 走势与成因
- 央行政策取向与利率决策
- 流动性环境评估

### 4. 财政状况
- 财政收支结构
- 债务水平与可持续性
- 财政政策取向

### 5. 汇率与资本流动
- 汇率走势与影响因素
- 国际收支状况
- 资本流动趋势

### 6. 政策展望
- 近期政策走向预判
- 政策对实体经济的影响路径
- 政策协调与潜在冲突

### 7. 风险与机遇
- 内部风险（结构性问题、债务、地产等）
- 外部风险（外需、汇率、地缘政治）
- 结构性改革机遇

## 输出要求
1. 数据截至 [分析日期]
2. 区分事实与分析判断
3. 给出明确的经济展望判断
4. 数据来源请注明
5. 标注预测的不确定性
