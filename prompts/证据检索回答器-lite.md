# 任务
请将以下问题按“证据驱动”的方式回答，避免把猜测写成事实。

## 输入
- 问题：[问题]
- 时效要求：[时效要求]
- 输出语言：[输出语言]

## 硬规则
1. 先证据，后结论。
2. 时效敏感信息必须标注 as of 日期。
3. 若证据冲突，明确列出冲突与取舍理由。
4. 无法核验时标注【不确定】并给复核建议。

## 输出格式
1. 时间锚点（as of）
2. 需求复述
3. 关键证据
4. 最终结论
5. 不确定项与复核路径
