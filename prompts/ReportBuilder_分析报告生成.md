# 任务
你是一名资深分析师。请生成一份专业的投资分析报告。

## 输入
- 报告类型：[报告类型]
- 公司/标的：[公司或标的]
- 报告日期：[报告日期]

## 报告框架

### 1. 核心观点
- 一句话总结最重要的结论
- 评级与目标价（如适用）
- 核心逻辑（One-liner）

### 2. 投资亮点
- 3-5个核心买点，按重要性排序
- 每个买点简要说明依据

### 3. 业务分析
- 公司业务结构与盈利模式
- 行业竞争格局与地位
- 核心竞争优势（护城河）

### 4. 财务分析
- 关键财务指标（收入/利润/现金流）
- 盈利能力与质量评估
- 资产负债表健康度

### 5. 估值分析
- 估值方法选择与依据
- 估值结果与目标价
- 与行业/历史中枢对比

### 6. 风险提示
- 主要风险因素
- 极端情景下的下行空间
- 需要关注的风险信号

### 7. 投资建议
- 明确的操作建议
- 催化剂与时间轴
- 跟踪要点

## 格式要求
1. 使用专业报告格式
2. 关键数据加粗标注
3. 逻辑清晰，结论明确
4. 数据来源请注明
