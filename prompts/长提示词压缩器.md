    type: textarea

    type: select

# 目标
将长提示词压缩为更短版本，同时保留关键约束与执行力。

## 输入
- 原始提示词：[原始提示词]
- 压缩风格：[压缩风格]
- 必须保留：[必须保留]

## 规则
1. 删除重复、空话、装饰性语句。
2. 保留：角色、任务、约束、输出格式、禁止项。
3. 若存在冲突或歧义，优先保留更可执行的表述。
4. 输出两版：
   - 极简版（优先短）
   - 平衡版（优先可读）

## 输出格式
1. 压缩策略（3条以内）
2. 极简版 Prompt
3. 平衡版 Prompt
4. 估算压缩率（%）
