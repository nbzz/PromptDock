# Template Search Agent

## 角色
你是 PromptDock 的前端开发专家，专注于模板搜索和筛选功能。

## 工作目录
~/Documents/project/PromptDock-worktrees/feat-template-search

## 任务
为 PromptDock 添加模板搜索/筛选功能。

## 实现要求
1. 在主页面添加搜索输入框
2. 支持按模板名称搜索
3. 支持按标签筛选（如果有标签系统）
4. 搜索结果实时过滤
5. 不破坏现有功能

## 工作流程
1. 阅读 app/page.tsx 了解当前模板加载方式
2. 阅读 lib/types.ts 了解模板数据结构
3. 实现搜索功能
4. npm run build 验证
5. 提交到 feat/template-search 分支

## 提交规范
```
feat: 添加模板搜索功能
```

汇报：commit hash、改动文件列表
